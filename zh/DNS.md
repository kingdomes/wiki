# DNS

## 概述

AFNOOI-MNT 维护着一组用于 DN42 网络内部解析的 DNS 服务，旨在为社区提供稳定、可靠的域名解析支持。

## 支持记录类型

本 DNS 服务基于 **Bind 9** 部署，目前支持以下标准解析记录类型：

| 记录类型 | 说明 | 常见用途 |
| :--- | :--- | :--- |
| **A** | IPv4 地址记录 | 将域名指向 IPv4 地址 |
| **AAAA** | IPv6 地址记录 | 将域名指向 IPv6 地址 |
| **CNAME** | 别名记录 | 将一个域名指向另一个域名 |
| **MX** | 邮件交换记录 | 指定邮件服务器地址 |
| **TXT** | 文本记录 | 存放 SPF、DKIM、验证信息等 |
| **NS** | 域名服务器记录 | 指定该域名的权威 DNS 服务器 |
| **PTR** | 反向解析记录 | 实现 IP 到域名的反向查询 |
| **SOA** | 起始授权记录 | 定义域名的管理信息（序列号、刷新时间等） |
| **NAPTR** | 命名权威指针记录 | 用于 ENUM（电话号码映射）和 SIP 路由解析 |

## 支持的域名后缀

本 DNS 服务支持以下域名区域的解析：

- `*.afn.dn42`
- `*.afn.neo`
- `*.4.2.2.1.4.2.4.0.tel.dn42`
- `*.3.2.2.1.4.2.4.0.tel.dn42`
- `*.dn42`（部分缓存转发）
- `*.neo`（部分缓存转发）
- `And more`

如需添加你的自定义域名或子域，请通查看联系联系 NOC 团队。
