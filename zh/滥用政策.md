# Abuse Policy

## 概述

AFNOOI-Network 致力于维护一个安全、稳定、可靠的网络环境。本滥用处理政策旨在明确针对网络滥用行为的处理流程，保障 DN42 社区及其他网络参与者的合法权益。

---

## 适用范围

本政策适用于以下情况：

- 使用 AFNOOI-Network 提供的网络资源（包括但不限于 DNS 服务、域名托管、隧道节点等）
- 通过 AFNOOI-Network 网络发起或中转的流量
- 与 AFNOOI-Network 存在对等互联（Peering）关系的网络

---

## 滥用行为的定义

以下行为被视为网络滥用，可能会触发本政策规定的处理流程：

| 类别 | 具体行为 |
| :--- | :--- |
| **恶意流量** | 发起 DDoS / DoS 攻击、端口扫描、暴力破解等 |
| **垃圾信息** | 发送未经请求的邮件（UCE / UBE）、SPAM 等 |
| **非法内容** | 传播恶意软件、钓鱼网站、侵犯版权或其他违法内容 |
| **网络干扰** | 伪造 IP 地址、BGP 路由劫持、DNS 欺骗等 |
| **资源滥用** | 超出合理使用范围的 DNS 查询、流量消耗、资源占用等 |

## 处理流程

收到滥用报告后，我们将按照以下步骤处理：

1. **受理确认**：在 24 小时内确认收到报告。
2. **初步核实**：根据提供的证据进行核查，判断是否属于本政策定义的滥用行为。
3. **处置措施**：根据严重程度，可能采取以下措施：
   - 发送警告通知
   - 限制相关服务或资源
   - 暂停或终止相关网络连接
   - 配合 DN42 社区或其他网络运营商进行联合处理
4. **反馈与记录**：向报告方反馈处理结果，并保留处理记录以备后续参考。
