## 拨号上网（Dial-Up）

AFNOOI-Network 在 DN42 网络中保留了一项最传统的拨号上网服务，致敬互联网初期的连接方式。

| 项目 | 内容 |
| :--- | :--- |
| **接入号码** | `+0424221316300` |
| **接入协议** | PPP / SLIP |
| **认证方式** | 无认证 / PAP / CHAP（可配置） |
| **调制解调器** | 支持 V.34、V.90、V.92 等标准协议 |
| **分配地址** | 由 Asterisk 通过 `modem` 通道分配 IP（实验性） |

### 使用方式

1. 使用ATA连接调制解调器。
2. 拨打接入号码 `+0424221316300`。
3. 调制解调器完成握手后，系统将建立 PPP 或 SLIP 会话。
4. 连接成功后，您将获得一个内网 IP 地址，并可访问 DN42 内部资源。

> ⚠️ 此服务为实验性质，受限于物理线路质量，实际速率通常不超过 56 kbps。不建议用于下载大文件或在线流媒体。

### 技术实现

- 基于 **Asterisk** 的 `app_chanisavail` 与 `modem` 通道
- 使用 **mgetty** 或 **pppd** 处理拨入会话
- 支持动态 IP 分配（via DHCP 或 RADIUS）

## 功能说明

### 传真服务

- 支持 **T.38** 传真协议（优先）及 **G.711** 透传
- 传真接收自动打印
- 支持 CNG 音检测并自动路由至传真处理流程

### 语音服务

- 基于 **Asterisk** 与 **PJSIP** 实现
- 支持 IPv4 / IPv6 双栈 ~~能不能通就不一定了~~
- 支持 **ENUM** 号码解析

### 短信服务

- `+042422131146` 支持 SMS 接收（通过 MicroSIP 客户端）

## 使用方式

### 呼入路由

所有以 `+04242213` 开头的号码将被路由至 AFNOOI-MNT 的 Asterisk 服务器，并根据号码分配规则转发至对应终端或应用。

### 呼出路由

- 通过 **ENUM** 查询目标号码，自动转换为 SIP URI 并完成呼叫
- 支持 DN42 内部号码直拨
