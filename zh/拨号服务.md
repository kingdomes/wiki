# 拨号服务

本页面介绍 AFNOOI-Network 在 DN42 网络中提供的 VoIP 拨号服务。

---

## 号码规划

所有号码均以 `+04242213` 为前缀，具体分配如下：

| 号码 | 功能 | Via | 说明 |
| :--- | :--- | :--- | :--- |
| `+042422131145` | 传真 / 电话 |Panasonic KX-FT982CN| 主号码，支持传真接收与语音通话 |
| `+042422131146` | 传真 / 电话 |MicroSIP 客户端 & FaxTalk| Windows MicroSIP & FaxTalk，支持传真、语音及短信接收|
| `+04242213110` | 语音通话 |普通座机| 仅支持语音接听，无传真功能 |
| `+0424221310086` | 话费查询 |IVR| 交互式语音应答（IVR）系统，可以查询账户余额、电话号码 |

---

## 功能说明

### 传真服务

- 支持 **T.38** 传真协议（优先）及 **G.711** 透传
- 传真接收自动打印
- 支持 CNG 音检测并自动路由至传真处理流程

### 语音服务

- 基于 **Asterisk** 与 **PJSIP** 实现
- 支持 IPv4 / IPv6 双栈 ~~能不能通就不一定了~~
- 支持 **ENUM** 号码解析

### 短信服务

- `+042422131146` 支持 SMS 接收（通过 MicroSIP 客户端）

---

## 使用方式

### 呼入路由

所有以 `+04242213` 开头的号码将被路由至 AFNOOI-MNT 的 Asterisk 服务器，并根据号码分配规则转发至对应终端或应用。

### 呼出路由

- 通过 **ENUM** 查询目标号码，自动转换为 SIP URI 并完成呼叫
- 支持 DN42 内部号码直拨

---

## 话费查询

拨打 `+0424221310086` 后，系统将播报当前账户余额信息。

> 查询系统目前仅持语音播报 ~~支持短信或邮件通知~~。

---

## 技术实现

- **PBX 核心**：Asterisk 20.6.0 + PJSIP
- **传真模块**：`res_fax` / `app_fax` + T.38 ~~+ 我的传真机~~
- **ENUM 解析**：基于 DN42 内部递归 DNS
- **短信功能**：`pjsip_messaging`

---

## 注意事项

- 所有服务仅限 **DN42 网络内部** 访问
- 请勿滥用本服务进行大量呼叫或传真轰炸
- 如有问题，欢迎通过 IRC 或邮件或电话传真联系 NOC
